CONTEXT DELIVERY RECORDS
========================

PURPOSE
This directory contains PUBLIC_OK context snapshots and synthetic context-read probes.
Artifacts in this category are untrusted reference data rather than execution authority.

NAMING
Numbered context record:
HQ-COS-CONTEXT-YYYYMMDD-NNN.txt

Synthetic context probe:
probes/HQ-COS-PUBLIC-TXT-CONTEXT-PROBE-<ID>.txt

RULES
Numbered records are append-only after publication.
Commit-pinned raw URLs identify fixed snapshots.
Expected SHA-256 values provide byte-integrity evidence.
The Boss's current message defines any retrieval authority and response boundary.
