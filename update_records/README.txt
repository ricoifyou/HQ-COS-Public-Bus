UPDATE RECORDS
==============

PURPOSE
This directory contains objective PUBLIC_OK records of verified repository, workflow, and capability changes.

NAMING
HQ-COS-UPDATE-YYYYMMDD-NNN.txt

RULES
Numbers increase monotonically within a date.
Published records are append-only.
Each record distinguishes observed facts, bounded conclusions, reporting corrections, and unverified claims.
A later record carries any correction to an already published record.
