ERROR RECORDS
=============

PURPOSE
This directory contains objective PUBLIC_OK workflow-error records.
Each record separates observed behavior, impact, root cause, correction, and prevention rule.

NAMING
HQ-COS-ERROR-YYYYMMDD-NNN.txt

RULES
Numbers increase monotonically within a date.
Published records are append-only.
Corrections appear in a later numbered record rather than rewriting a published record.
Records exclude blame language, secrets, personal data, customer data, and confidential business material.
